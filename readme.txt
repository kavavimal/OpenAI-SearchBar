Add the ChatGPT Search Bar into your website pages without having to manually edit your website's source code.

== Description ==

- The ChatGPT AI Search Bar plugin gives access to a feature that allows you to Add a powerful AI search bar to your website using shortcodes.

- This shortcode is a simple line of code that can be easily added to any page or post, without the need for you to manually edit your website's source code.

- This convenient plugin feature makes it easy for you to incorporate the ChatGPT AI search bar into your website, providing your visitors with a more accurate and relevant search experience.

- We are using 3rd party OpenAI Services to fetch results : https://api.openai.com/v1/chat/completions

== Key Features ==

- Once you upload and activate the ChatGPT AI search bar plugin, you will have access to a feature that allows you to add a powerful AI search bar to your website using a shortcode.

== Installation ==	

Installation Via WordPress Admin Area

1. Log into your WordPress admin area.
1. Go to Dashboard » Plugin.
1. Click on Add New Plugin button at top.
1. Search for “OpenAI : AI Search Bar”.
1. Now click on “Install Now” button of the “OpenAI : AI Search Bar” plugin.
1. Click on Activate once the plugin is installed.

Installation Using FTP

1. Download the openai-search-bar.zip’ file.
1. Extract Zip file.
1. Using your FTP program, upload the non-zipped plugin folder into the “/wp-content/plugins/” folder.
1. Activate the plugin through the ‘Plugins’ menu in WordPress.
